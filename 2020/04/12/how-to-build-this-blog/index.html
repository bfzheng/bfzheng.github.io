How to build this blog

Posted on 2020-04-12

摘要

本篇主要介绍我是如何搭建博客网站的过程，网站主要使用Hexo+Github pages搭建，具体流程如下。

博客搭建

1. Create a new project

在 github 创建一个新的 repo，名称为：${your github name}.github.io（下面我将使用test.githug.io来代替），然后把项目 clone 到本地。
此处需要注意的点有两个：

  • 为了开启github pages，repo命名需要按照上述所示格式；
  • 每个github账户只允许创建一个这样的repo。

2. Install Hexo

3. Hexo init project

4. Run local server

5. Set blog infor

6. Set use git to deploy

7. Create a new post

8. Generate and deploy

9. Change blog theme

10. Create a new page
